### 🔮 Overview

> This repository contains the core of my ALX Software Engineering training focused on low-level programming using the C language. Each project folder explores a new chapter in the epic of understanding machines from the ground up.

---

## 🗂️ Folder-by-Folder Adventure

---

### `0x00-hello_world` — _"The Awakening"_

💬 **Intro to C compilation steps** (Preprocessing → Compiling → Assembling → Linking), and the classic `puts` vs `printf`.

📚 **Learn More**:
- [Compilation Process](https://www.geeksforgeeks.org/compilation-process-c-program/)
- [Hello World Explained](https://fresh2refresh.com/c-programming/c-hello-world/)

---

### `0x01-variables_if_else_while` — _"The Branching Path"_

🔁 **Master flow control**: `if`, `else`, `while`, comparisons, logic. Also covers ASCII and basic RNG.

📚 **Learn More**:
- [Control Flow in C](https://www.programiz.com/c-programming/if-else)
- [Random Numbers](https://www.geeksforgeeks.org/rand-and-srand-in-ccpp/)

---

### `0x02-functions_nested_loops` — _"The Function Forge"_

🔧 Craft functions like spells. Learn nested loops, scope, and modular code structure.

📚 **Learn More**:
- [Functions](https://www.tutorialspoint.com/cprogramming/c_functions.htm)
- [Nested Loops](https://beginnersbook.com/2014/01/c-nested-loops/)

---

### `0x03-debugging` — _"The Bug Hunter's Toolkit"_

🕵️ Practice identifying and fixing logic bugs. Hands-on with `gdb`, test cases, and conditional checks.

📚 **Learn More**:
- [Debugging in C](https://www.cs.cmu.edu/~gilpin/tutorial/)
- [GDB Guide](https://darkdust.net/files/GDB%20Cheat%20Sheet.pdf)

---

### `0x04-more_functions_nested_loops` — _"More Loops, More Power"_

🌀 Practice advanced loops and conditions with exercises like FizzBuzz, prime factors, and printing shapes.

📚 **Learn More**:
- [FizzBuzz Logic](https://dev.to/mindninjax/fizzbuzz-in-c-1ljc)
- [Loop Patterns](https://www.sanfoundry.com/c-programming-examples-loops/)

---

### `0x05-07-pointers_arrays_strings` — _"The Memory Labyrinth"_

📌 Deep-dive into pointers, memory addresses, strings, and buffer manipulation.

📚 **Learn More**:
- [Pointer Basics](https://boredzo.org/pointers/)
- [C Strings](https://www.learn-c.org/en/Strings)

---

### `0x08-recursion` — _"Calling Yourself"_

🌀 Learn the mystical art of recursion — functions that call themselves. Reverse strings, factorials, power functions.

📚 **Learn More**:
- [Recursion Visualized](https://visualgo.net/en/recursion)
- [C Recursion](https://www.programiz.com/c-programming/c-recursion)

---

### `0x09-static_libraries` — _"Archives of Code"_

📚 Build `.a` static libraries using `ar`, and compile projects with reusable components.

📚 **Learn More**:
- [Static Libraries](https://www.geeksforgeeks.org/static-vs-dynamic-libraries/)

---

### `0x0A-argc_argv` — _"Talking to Your Program"_

🗣️ Parse input from the command line. Count arguments and process strings.

📚 **Learn More**:
- [argc & argv](https://www.geeksforgeeks.org/command-line-arguments-in-c-cpp/)

---

### `0x0B-0C-malloc_free` — _"Dynamic Dimensions"_

⚙️ Learn heap memory allocation and deallocation with `malloc` and `free`. Detect and prevent memory leaks.

📚 **Learn More**:
- [Valgrind Basics](https://valgrind.org/)
- [malloc in C](https://www.geeksforgeeks.org/dynamic-memory-allocation-in-c-using-malloc-calloc-free-and-realloc/)

---

### `0x0D-preprocessor` — _"The Meta Layer"_

🧩 Explore macros, header guards, and compile-time directives.

📚 **Learn More**:
- [Preprocessor Directives](https://gcc.gnu.org/onlinedocs/cpp/)

---

### `0x0E-structures_typedef` — _"Blueprints of Data"_

🧱 Build custom data types using `struct` and simplify with `typedef`.

📚 **Learn More**:
- [C Structs](https://www.programiz.com/c-programming/c-structures)

---

### `0x0F-function_pointers` — _"Pointing to Behavior"_

📌 Use functions as data. Pass them, store them, invoke them dynamically.

📚 **Learn More**:
- [Function Pointers](https://www.geeksforgeeks.org/function-pointer-in-c/)

---

### `0x10-variadic_functions` — _"Flexible Arguments"_

📦 Use `va_list` to accept an undefined number of function arguments, like `printf()` does.

📚 **Learn More**:
- [Variadic Functions](https://en.cppreference.com/w/c/variadic)

---

### `0x12-13-singly_linked_lists` — _"Chains of Memory"_

🔗 Traverse, modify, and build forward-only lists. Practice with dynamic structures.

📚 **Learn More**:
- [Singly Linked List](https://www.geeksforgeeks.org/data-structures/linked-list/singly-linked-list/)

---

### `0x14-bit_manipulation` — _"Binary Sorcery"_

💡 Shift bits, flip bits, and analyze values at the lowest level.

📚 **Learn More**:
- [Bit Tricks](https://graphics.stanford.edu/~seander/bithacks.html)

---

### `0x15-file_io` — _"Files, Meet System Calls"_

🗃️ Use `open`, `read`, `write`, and `close` to work with file descriptors like a UNIX master.

📚 **Learn More**:
- [System File I/O](https://man7.org/linux/man-pages/man2/open.2.html)

---

### `0x17-doubly_linked_lists` — _"Bidirectional Chains"_

🔁 Master lists that can go forward and backward. Understand node insertions/deletions.

📚 **Learn More**:
- [Doubly Linked Lists](https://www.geeksforgeeks.org/doubly-linked-list/)

---

### `0x18-dynamic_libraries` — _"Living Code"_

🧬 Build `.so` libraries. Load them during runtime. Create flexible applications.

📚 **Learn More**:
- [Dynamic Linking](https://linuxhint.com/shared-libraries-linux/)

---

### `0x1A-hash_tables` — _"Fast Lookups"_

📇 Build custom hash tables. Handle collisions and optimize key-based retrieval.

📚 **Learn More**:
- [Hash Tables in C](https://www.programiz.com/dsa/hash-table)

---

### `0x1C-makefiles` — _"The Build Automator"_

🛠️ Automate builds using `make`. Write your own rules and recipes.

📚 **Learn More**:
- [Makefile for Beginners](https://makefiletutorial.com/)

---

### `0x1E-search_algorithms` — _"Finding the Needle"_

🔍 Implement Linear and Binary Search algorithms. Understand efficiency and Big-O.

📚 **Learn More**:
- [Searching Algorithms](https://visualgo.net/en/search)
